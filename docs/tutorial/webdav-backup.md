# WebDav 备份教程

[[toc]]

本应用支持云备份，采用 WebDav 协议。所有支持 WebDav 的云盘均可使用。

## 配置步骤

1. 注册并登录您选择的 WebDav 云盘服务
2. 进入账户设置，找到安全选项或第三方应用管理
3. 添加一个新的第三方应用，获取服务器地址、用户名和密码
4. 打开本应用，进入 **我的 → 备份与恢复 → WebDav 设置**
5. 填入获取的服务器地址、账号和密码
6. 创建子文件夹选项保持默认即可
7. 设置完成后手动执行一下备份，验证是否成功
8. 恢复时选择想要恢复的备份文件

## 自动备份说明

- 设置好备份之后每次退出 App 会自动进行备份
- WebDav 同一天的备份会覆盖，不同日期的备份不会覆盖
